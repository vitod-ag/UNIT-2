SASS

Pre-processore css

SASS è un pre-processore CSS che applica a CSS3 strutture di programmazione (variabili, cicli, condizioni...)

In SASS è possibile utilizzare anche variabili complesse con coppie chiave/valore, simili agli oggetti Javascript
